What is Athena

You've read the books.
Now let them work for you.

Every Product Manager knows this feeling: it's 4 PM on a Thursday and you need to write a PRD, design an experiment, or prepare a competitive battlecard — and you're starting from scratch. Every. Single. Time.

Athena eliminates that gap. By embedding 65 deeply structured PM skills and 36 orchestrated workflows into your Antigravity IDE, Athena turns proven product frameworks into executable, repeatable processes.

"You don't just get advice — you get a guided, step-by-step co-pilot that asks the right questions, applies the right frameworks, and produces professional-grade deliverables in minutes."

Get Started

From zero to PM co-pilot
in a few minutes

No prior coding experience required. Follow these steps and you'll have Athena running in your IDE before your next meeting.

1
Step One

Install Antigravity IDE

Antigravity is Google's free, AI-first development platform — your command centre for running Athena. It's available for Mac, Windows, and Linux, and takes about 5 minutes to install.

🍎 Download for Mac 🪟 Download for Windows 🐧 Download for Linux
1
Open the downloaded .dmg file from your Downloads folder.
2
Drag the Antigravity icon to your Applications folder.
3
Open Antigravity from Applications. Click through the welcome screens — choose "Start Fresh" unless you want to import VS Code settings.
1
Double-click the downloaded .exe installer.
2
Follow the installation wizard. Accept defaults — they're fine.
3
Launch Antigravity from the Start menu. Choose "Start Fresh" on first run.

For Ubuntu/Debian — run these commands in your terminal:

# Add the Antigravity repository
curl -fsSL https://antigravity.google/linux/gpg | sudo apt-key add -
sudo add-apt-repository "deb [arch=amd64] https://antigravity.google/linux/deb stable main"

# Install
sudo apt update && sudo apt install antigravity

# Launch
antigravity
2
Step Two
✦ Optional — but recommended

Connect a Google Account

Signing in with a personal Gmail account unlocks Gemini 3 Pro with generous rate limits — completely free. This gives Athena more power to reason deeply, write long deliverables, and scrape live market data for you.

1
On first launch, Antigravity will prompt you to sign in. Click "Sign in with Google".
2
Use a personal Gmail account (not a Google Workspace / company account, as policies may restrict access).
3
Once signed in, select Gemini 3 Pro as your model from the model selector. No API key needed — it's pre-integrated.
💡 Don't have a Gmail account? No problem — Antigravity works without signing in too, or you can bring your own Claude / OpenAI API key via the model settings panel.
3
Step Three

Open Athena as a Project

Athena lives inside a GitHub repository. You'll clone it (download a copy) to your computer, then open it as a project in Antigravity. That's it — no installation, no configuration files to edit.

In your terminal (or Antigravity's built-in terminal), run:

git clone https://github.com/manticorevault/athena_pm_skills.git
cd athena_pm_skills
1
If you don't have Git installed, download it here first (it's a one-click installer).
2
In Antigravity, go to File → Open Folder and select the athena_pm_skills folder you just cloned.
3
Antigravity will load the project. You'll see the .agents/ folder — that's where all of Athena's intelligence lives.
🔒 Not a developer? You can also download the repository as a ZIP file from GitHub (click the green "Code" button → "Download ZIP"), unzip it, and open the folder in Antigravity.
4
Step Four

Start a Conversation with Athena

With the project open, open a new chat in Antigravity's Agent panel. You're now talking to Athena. Type any slash command below and press Enter — Athena will guide you through every step.

# Try your first command:

/discover A mobile app that helps remote teams align on priorities

# Or jump straight into a PRD:
/write-prd User notifications for our SaaS dashboard
✨ Athena will ask you clarifying questions, apply frameworks automatically, and produce a full, structured deliverable — ready to share with your team or stakeholders.
All 36 Workflows

Every PM task.
One slash command.

Browse Athena's complete command library. Each workflow chains multiple frameworks together and produces a professional, shareable deliverable.

🔍 Product Discovery
/discover
Full discovery cycle — ideation through assumption mapping to experiment design. Athena's flagship 15–30 min sprint.
/brainstorm
Generate 10+ product ideas from PM, Designer, and Engineer perspectives for any product.
/triage-requests
Transform a messy backlog of feature requests into a structured, scored, and actionable priority list.
/interview
Prepare a customer interview script — or synthesise a transcript into themes, quotes, and opportunities.
/setup-metrics
Design a full metrics dashboard: North Star, input metrics, health metrics, and alert thresholds.
♟️ Product Strategy
/strategy
Comprehensive strategy via the 9-section Strategy Canvas — vision, competitive moat, key metrics, and strategic bets.
/business-model
Explore Lean Canvas, Business Model Canvas, or Value Proposition frameworks section by section.
/market-scan
SWOT, PESTLE, Porter's Five Forces, and Ansoff Matrix in one comprehensive scan.
/pricing
Design a pricing strategy — freemium, tiered, usage-based, or enterprise — with test designs.
⚙️ Execution
/write-prd
Full PRD with problem definition, user stories, success metrics, and launch criteria.
/write-stories
Break a feature into dev-ready backlog items with acceptance criteria and definitions of done.
/sprint
Plan a sprint, run a retro, or generate release notes — adapts to your current phase.
/pre-mortem
Identify what could go wrong before launch — across technical, market, and user dimensions.
/meeting-notes
Paste a transcript → clean, shareable notes with decisions and action items in seconds.
/stakeholder-map
Map stakeholders on a Power × Interest grid with tailored communication plans.
🔬 Market Research & 📈 Analytics
/competitive-analysis
Enhanced with live web scraping — Athena pulls real competitor pricing and features before analysing.
/research-users
Build personas, segment users, and map the customer journey from your research data.
/analyze-feedback
Paste NPS responses or support tickets — get structured, actionable themes from any volume.
/analyze-test
A/B test results with statistical significance, sample validation, and a clear ship/extend/stop call.
/write-query
Describe what you need in plain English → production-ready SQL for BigQuery, PostgreSQL, and more.
🎯 Go-To-Market & 🌱 Growth
/plan-launch
Full GTM playbook: beachhead segment, ICP, messaging, channels, and launch plan.
/battlecard
Sales-ready competitive battlecard: positioning, objection handling, and win strategies.
/growth-strategy
Map viral, content, paid, and sales-assisted growth loops with metrics for each.
/north-star
Define your North Star Metric and input metrics — validated against best practices.
🛠️ PM Toolkit

PM resume review against 10 best practices — structure, impact metrics, keywords, and coaching.
/tailor-resume

/proofread
Surgical precision editing — fixes grammar, logic, and flow without rewriting your voice.
